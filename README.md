# 🧠 Jogo da Memória em JavaScript

Projeto desenvolvido como parte de um desafio prático para reforçar conceitos de desenvolvimento web utilizando **HTML, CSS e JavaScript puro**.

---

## 🎯 Entendendo o Desafio

### 🎯 AGORA É A SUA VEZ

Chegou a hora de colocar em prática tudo aquilo que você aprendeu durante esse projeto.

### 👨‍💻 OBJETIVO

Sua missão foi:

* Reproduzir passo a passo o projeto apresentado em aula
* Implementar melhorias e personalizações
* Publicar o projeto no GitHub
* Compartilhar o repositório como entrega final

---

## 🚀 Funcionalidades

* 🎴 Cartas com pares de emojis
* 🔀 Embaralhamento automático
* ⏱ Timer regressivo
* 🎯 Contador de tentativas
* 🏆 Sistema de vitórias
* 📈 Níveis com dificuldade progressiva
* 🔁 Botão de reset do jogo
* 🔒 Bloqueio de jogadas simultâneas
* 🎉 Detecção de vitória

---

## 🎮 Como Jogar

1. Clique na tela para iniciar
2. Vire duas cartas por vez
3. Encontre todos os pares
4. Termine antes do tempo acabar

---

* HTML5

* JavaScript
